### How to use?

1. Make sure to have your API running at http://localhost:1337
2. `git clone git@github.com:bernardodiasc/poc-react-context.git`
3. Go to `shared` 
4. `yarn install`

### Gatsby

1. Go to `gatsby`
2. `yarn install`
3. `yarn dev`
4. Open http://localhost:8000/

### NextJS

1. Go to `nextjs`
2. `yarn install`
3. `yarn dev`
4. Open http://localhost:3000/
